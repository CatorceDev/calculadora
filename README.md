# 🧮 Calculadora JavaScript

Una **calculadora interactiva** desarrollada en JavaScript, HTML y CSS, que permite realizar operaciones básicas como suma, resta, multiplicación y división. Incluye funcionalidades de **borrar dígito**, **limpiar pantalla** y **repetir operaciones con el botón igual** al estilo de las calculadoras tradicionales.

---

## 🔹 Funcionalidades

- ✅ Suma, resta, multiplicación y división  
- ✅ Mostrar resultados en tiempo real  
- ✅ Borrar el último dígito  
- ✅ Limpiar toda la pantalla  
- ✅ Repetir la última operación con `=` varias veces  
- ✅ Compatible con navegadores modernos  

---

## 💻 Tecnologías

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
